Foire Aux Questions

Problème d'imprimante+

Vérifiez que l'imprimante est bien branchée et allumée. Vérifiez les pilotes et assurez-vous qu'elle est définie comme imprimante par défaut. Si le problème persiste, vérifiez le niveau d'encre ou de toner.

Réinitialiser mon mot de passe+

Cliquez sur "Mot de passe oublié" sur la page de connexion. Entrez votre email et suivez les instructions envoyées. Vérifiez votre dossier spam si vous ne recevez pas l'email.

Problème de connexion Wi-Fi+

Redémarrez votre routeur et votre ordinateur. Vérifiez que vous êtes connecté au bon réseau et que le mot de passe est correct. Essayez de vous rapprocher du routeur.

Installation de logiciels+

Contactez l'administrateur pour obtenir les droits d'installation. Téléchargez uniquement depuis les sites officiels ou le portail d'entreprise. Ne installez jamais de logiciels non approuvés.

Ordinateur trop lent+

Fermez les applications inutilisées. Vérifiez l'espace disque disponible (minimum 10% recommandé). Effectuez un redémarrage. Si le problème persiste, contactez le support.

Erreur de connexion VPN+

Vérifiez votre connexion internet. Assurez-vous d'utiliser les bonnes identifiants VPN. Redémarrez l'application VPN. Si l'erreur persiste, créez un ticket avec le code d'erreur affiché.

Sauvegarde des données+

Sauvegardez régulièrement vos fichiers importants sur le serveur réseau ou le cloud d'entreprise. Ne stockez pas de données sensibles sur votre disque local uniquement.

Problème avec Outlook/Email+

Vérifiez votre connexion internet. Redémarrez Outlook. Vérifiez que votre boîte mail n'est pas pleine (limite: 2GB). Si les problèmes persistent, recréez le profil Outlook.

Écran bleu (BSOD)+

Notez le code d'erreur affiché. Redémarrez l'ordinateur. Si le problème se répète, créez un ticket urgent avec le code d'erreur et la fréquence du problème.

Périphérique USB non reconnu+

Essayez un autre port USB. Redémarrez l'ordinateur. Vérifiez sur un autre ordinateur si le problème vient du périphérique. Mettez à jour les pilotes USB.

Vous ne trouvez pas votre réponse ?

Notre équipe de support est disponible pour vous aider.
